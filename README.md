请所有贡献者，以以下格式发布翻译：

```
## 比赛编号

|A [难度A]|B [难度B]|C [难度C]|……|
|:-:|:-:|:-:|:-:|
|[英文原标题 \| 中文翻译标题](/problem/比赛编号/字母编号)（翻译过的标签）|其余同理|……|……|
```
未有过翻译的字母编号跳过即可。

例如：

```
## 1

|A [1000]|C [难度]|
|:-:|:-:|
|[Theatre Square \| 剧院广场](/problem/1/A)（数学）|[CF1C原标题 \| 翻译过的awawawa](/problem/1/C)（TAG1，TAG2，TAG3）
```

## 1

|A [1000]|C [难度]|
|:-:|:-:|
|[Theatre Square \| 剧院广场](/problem/1/A)（数学）|[CF1C原标题 \| 翻译过的awawawa](/problem/1/C)（TAG1，TAG2，TAG3）

（标签使用全角逗号隔开 `，`）

请注意，当你在 `/docs/zh/` 目录下更新内容时（题目列表除外），请在开头加上：

```
---
author: 你的GitHub用户名
---
```

注意 `:` 与 用户名 中间隔了个空格。

如果你是**修改别人的内容**，则这样改：

```
---
author: 之前填在这的用户名(可能有多个),你的GitHub用户名
---
```

注意 `,` 两边**都没有空格**。

**还有很重要的一件事：**

如果你是发布了一个没有人发布过的翻译（也就是在 `/docs/zh/` 创建了新的文件），请在 **Pull Request** 中同步也更新 `/docs/en/` 的页面，塞上原文即可（不要写author）。

`/docs/en/` 的题目列表，按照以下格式，也必须更新：

```
## 比赛编号

|A [难度]|B [难度]|...|
|:-:|:-:|:-:|
|[原题目标题](/en/problem/比赛/A) (标签)|以此类推|...|
```
## 55555

|A [888888]|B [888888888]|...|
|:-:|:-:|:-:|
|[Aba Aba QAQ](/en/problem/55555/A) (dp, ddp, dddp, dddfs)|以此类推|...|

请注意标签用 `, ` 隔开（前面无空格，后面有）
