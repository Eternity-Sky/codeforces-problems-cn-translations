# C. 古老的伯兰马戏团

## 题目描述

现在贝兰的所有马戏团都有一个直径为13米的圆形竞技场,但过去的情况有所不同.

在古老的马戏团中的贝兰竞技场被塑造为常规(等角)多边形,大小和取角数量可能因马戏团而异. 竞技场的每一个角落都有一个特殊的柱子,柱子之间的绳子被打出标志了竞技场边缘.

最近来自伯兰的科学家发现了古代马戏团竞技场的残骸. 他们只发现了三根柱子,其他被当时所摧毁.

你得到这三根支柱的坐标。 找出竞技场能拥有的最小区域.

## 输入格式

输入文件由三行组成,每行包含一对数字QQ坐标的支柱. 任何坐标以绝对值计算不超过1000,在小数点后最多给出6位数.

## 输出格式

输出出古代竞技场最小的可能区域. 这个数字应该准确到小数点后至少6位数. 可以保证,最佳多边形中取角的数量不会大于100个.

## 样例

### 输入

```
0.000000 0.0000001.000000 1.0000000.000000 1.000000
```

### 输出

```
1.00000000
```
