# B. 电子表格

## 题目描述

在通俗的电子表格系统(例如Excel)中,使用下列列的编号. 第一列有A号,第二列——B号等,至以Z为标志的第26列. 然后是两个字母的编号: 第27栏有AA,28——AB, 第52栏以AZ为标志. 在ZZ之后,会遵循三个字母编号等.

各行以整数为标志,从1. 单元格名是列和行号的缩合. 例如,BC23是第55栏第23行中单元格的名称.

有时会使用另一个编号系统:RXCY,其中X和Y为整数,以尊重的方式显示列和行数. 例如,R23C55就是前一个例子的单元格.

您的任务就是写出一个读取给定的单元格坐标序列的程序,并且按照另一个数字系统的规则制作每个项目.

## 输入格式

输入的一行包含整数n (1 ≤ n ≤ 10 5),即测试中的坐标数. 然后沿着n行走,每一行都包含坐标. 所有坐标都正确,没有列和/或行号大于10 6的单元格.

## 输出格式

写n行，每行应在另一个记数系统中包含一个单元格坐标。

## 样例

### 输入

```
2R23C55BC23
```

### 输出

```
BC23R23C55
```
